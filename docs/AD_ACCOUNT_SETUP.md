## Phase 1: コンバージョン設定

### 1.1 コンバージョンアクションの作成

Google Ads UI：**ツールと設定 → 測定 → コンバージョン → 新しいコンバージョンアクション**

- アクションのソース: **ウェブサイト**
- ウェブサイト: `manshitsu.homes`
- 「コードを使用して手動でコンバージョンアクションを作成」を選択
- 設定:
  - **コンバージョン名**: `LINE登録（満室収益物件LP）`
  - **カテゴリ**: 見込み顧客（登録）
  - **値**: 各アクションに同じ値を使用 → **¥3,000**（見込みLTV÷10程度）
  - **カウント**: 1
  - **コンバージョン計測の期間**: 30日
  - **エンゲージメントビュースルー**: 3日
  - **アトリビューション**: データドリブン（推奨）
  - **「コンバージョン」列に含める**: ✅ オン

### 1.2 タグのsend_toラベルを取得

コンバージョン作成後に表示される「Google タグ」セクションで以下のようなコードスニペットが出る：

```js
gtag('event', 'conversion', {'send_to': 'AW-18072645905/XXXXXXXXXXXXXXXXXX'});
```

この `AW-18072645905/XXXXXXXXXXXXXXXXXX` の**コンバージョンラベル部分**をコピーしておく。

### 1.3 LP側のコンバージョン発火コードを更新

**作業内容**: 取得したコンバージョンラベルを使って、`line_cta_click` 発火時に同時にコンバージョンを送信するよう LP を更新します。

→ ラベルを取得したら、Claude Code に「コンバージョンラベルは `AW-18072645905/xxx` です、コンバージョン発火コードを追加して」と依頼してください。LP の7箇所（Hero/MidCTA/Guide/BottomCTA/ExitPopup/MailForm/FloatingCTA）に一括で反映します。

- デバイス別: スマホの方がCPA安い傾向 → PC入札 -20% などのテスト

- 地域: 日本全域で回す中で、首都圏の反応が良ければ +20% テスト

## 付録A: よくある落とし穴

1. **地域限定設定の罠**: 「宮城県」に地域限定すると検索量が1/50以下になる。必ず「日本全域」で配信し、コピーで宮城をアピールすること。
2. **広告停止の放置**: 支払い問題で停止した後、再開時に予算を戻し忘れるケース多し。
3. **「不動産投資」の誘惑**: 検索量33,100の誘惑に負けない。学習層はCVR低く予算食い潰される。
4. **サイトリンクを設定せずに配信**: CTRが半減する。必ずPhase 6を設定してから配信開始。
5. **広告文の「無料」連呼**: Googleの広告ポリシーでリジェクト率が上がる。「無料」は1見出し1本まで。
